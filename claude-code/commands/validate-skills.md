# 验证 Skills

验证所有保存的 skills 内容是否仍然准确。

## 工作流

### 1. 扫描所有 skill 文件

```bash
find ~/.claude/skills -name "*.md" -type f
```

### 2. 提取代码块

从每个文件提取所有代码块。

### 3. 验证检查清单

```markdown
## 验证报告

### 代码示例
- [ ] 所有代码示例语法正确
- [ ] 所有模式可以编译/运行

### API 准确性
- [ ] API 方法名正确
- [ ] 参数签名正确

### 版本兼容性
- [ ] 标记的版本要求准确
- [ ] 没有使用已废弃的 API

### 完整性
- [ ] 没有缺少关键步骤
- [ ] 示例代码完整可用
```

### 4. 报告问题

```markdown
### 发现的问题

1. **{file}**: {问题描述}
   - 行号: {line}
   - 建议: {fix}

2. ...
```

### 5. 提供修复选项

```
发现 N 个问题。

1. 自动修复所有
2. 逐个审查修复
3. 只显示报告
```

## 验证后

添加验证标记到文件:

```markdown
<!-- validated: YYYY-MM-DD | status: pass|issues | by: claude -->
```
