# Skills 健康报告

显示所有保存的 skills 的使用统计和健康状态。

## 工作流

### 1. 扫描所有 skill 文件

```bash
ls ~/.claude/skills/*.md 2>/dev/null
```

### 2. 提取反馈标记

从每个文件提取 `<!-- feedback: ... -->` 标记。

### 3. 生成报告

```markdown
## Skills 健康报告

生成时间: {当前日期}

### 按技术栈

| 技术栈 | 模式数 | 平均成功率 | 状态 |
|-------|-------|-----------|------|
| flutter | 5 | 85% | ✅ |
| swiftui | 3 | 60% | ⚠️ |

### 模式详情

| 模式 | 使用 | 成功 | 失败 | 成功率 | 状态 |
|------|-----|------|------|-------|------|
| Flutter 下拉刷新 | 10 | 9 | 1 | 90% | ✅ 健康 |

### 需要关注

- ❌ 成功率 <50% 的模式
- ⚠️ 成功率 50-70% 的模式

### 建议

1. 修复失败率高的模式
2. 审查边缘情况
3. 移除长期未使用的模式
```

## 健康状态定义

| 成功率 | 状态 | 建议 |
|-------|------|------|
| > 90% | ✅ 健康 | 无需动作 |
| 70-90% | 🔶 良好 | 可选审查 |
| 50-70% | ⚠️ 需审查 | 检查边缘情况 |
| < 50% | ❌ 问题 | 需修复或移除 |
